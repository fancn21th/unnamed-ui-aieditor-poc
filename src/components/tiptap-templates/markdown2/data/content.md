# Markdown2 Editor

Welcome to the **Markdown2** template. It keeps the Simple toolbar and writes content as markdown.

> You can edit with buttons or keyboard shortcuts like `⌘+B`.

## Formatting

- **Bold**
- *Italic*
- ~~Strike~~
- `Inline code`
- <u>Underline</u>
- X<sup>2</sup> and H<sub>2</sub>O
- <mark style="background-color: var(--tt-color-highlight-yellow)">Highlight</mark>

## Lists

1. Ordered list
2. With nested bullet
   - Nested bullet

- [x] Task completed
- [ ] Task pending

## Blockquote and code block

> Blockquote content with [a link](https://tiptap.dev).

```ts
const framework = "tiptap"
console.log(framework)
```
